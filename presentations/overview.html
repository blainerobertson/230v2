Teaching Presentations

What

During the course each student is expected to create a 7-minute presentation on a topic from the supplied topics list.

How

This presentation can be created using presentation software like PowerPoint or Keynote, or using a web-based program like Prezi, Haiku Deck, Google Slides, a self-built web page or something similar.

Where

The presentation will be given at or near the beginning of class on the scheduled date.

When

The presentation will be given in class on the date scheduled. After signing up, add a reminder to your own calendar with an alert far enough in advance that you do not forget your presenation.

Review

Review the Teaching Presentation Grading Matrix found in this module within I-Learn.

Sign Up for Your Presentation

Sign-up for a topic using the Teaching Presentation Sign-up link in this module.
