Browser Tools

There are several browser tools that we will add to Chrome and FireFox for use in the class. Note: You will have your regular version of Chrome and a Brackets developer version. You will want to add the tools to both.

Web Developer

The Web Developer tool by Chris Pederick is a fantastic tool for validation, document outline checking and hundreds of other development tasks. This is a must have tool!

  • Web Developer for Chrome
  • Web Developer for FireFox

WAVE

The WAVE tool is an accessibility checker developed by Utah State University. A great tool for checking web pages to meet accessibility standards.

  • Wave for Chrome
  • Wave for FireFox

Color Picker

These are separate tools for Chrome and FireFox that do the same thing - identify and copy colors that you find on web sites. Great for building color palettes for sites.

  • Chrome Eye Dropper
  • FireFox Colorzilla

YSlow

The YSlow tool checks the weight of web pages when they load into a browser by determining the size of all files (html, css, javascript, images, etc...) that must be downloaded. For our class the maximum weight is 400kb. This tool is currently only available for Chrome - YSlow for Chrome.

LastPass

This tool is not required but is suggested. With security becoming more of a concern daily a good password manager is a tool that you should have. LastPass is such a tool. I recommend it.

  • LastPass Download Page (All major browsers)
