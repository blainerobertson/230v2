Weather Underground API

Register and Get a Key

wunderground.com (Weather Underground) is a weather site that provides weather data to many sites across the web. They have a free service that has limited use for developers and small web sites (such as ours). You will be doing many of the same tasks with wunderground.com that were demonstrated in the jQuery: Beyond the Basics videos you watched in JQuery and JSON activity.

  1. Go to wunderground.com and register for a Basic (free) account.
  2. Once registered, login using your username and password.
  3. Apply for an API key for a "Stratus Developer" plan (this should be free) for your own web site.
  4. Make sure you read the "API Terms of Use" agreement, particularly the "Conditions and Restrictions, item a" and "Rate Limits; Overage Policy".
  5. Once you have received your API key, make a copy of it and store it where you can access it easily in your own computer. DO NOT share it with anyone. You are responsible for it.

Look at the Code Samples

This section is to learn how the API works and how you follow its rules in order to get data from it. It is critical that you pay attention to details as you look at these examples.

  1. Visit the Code Samples page to look at the options available for accessing the data from the API.
  2. Visit the geolookup page to review the type of data that is returned from a geolookup query.
    1. Scroll all the way to the bottom and examine the various options for getting geolocation data: including: name, zip code, longitude and latitude.
  3. Visit the conditions page to review the type of data that is returned from a conditions query.
  4. Visit the Data Features: URL Format page to see how multiple data types can be requested at the same time.
