Getting Started With GitHub

What is GitHub?

GitHub is a web site that allows users to store and share files, applications and projects. The technology behind GitHub can be daunting, but for our class we will attempt to keep things simple.

A recent implementation by the folks are GitHub was the introduction of web repositories. These are simple web sites for promoting projects. The advantage is that it is free, it supports HTML5, CSS3 and JavaScript and provides a "https" protocol for the site. Perfect for our class.
