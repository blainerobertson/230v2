Introduction to HTML

Hypertext Markup Language (HTML) is the primary language of the World Wide Web. It is what you will use to build the content of your web pages and indicate the organization of that content to the browser. In order to get a fast start with HTML you will watch and follow along with the Introduction to HTML video course (107 minutes).

As you watch the video course keep in mind that you should download the files and follow along with the course in order to better understand and master the material - please don't just watch it!

Go to the Introduction to HTML video course

Validation

Lesson 5 in the video course discusses and demonstrates validating your code to make sure it is accurate. Pay close attention to this as every web page you produce MUST be valid HTML5 code, including the documents for this week!

Read the Book Too
