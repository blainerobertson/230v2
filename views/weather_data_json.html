What is JSON?

Like so many other things, JSON is an acronym that stands for JavaScript Object Notation. This is geek speak for data that is organized in "name - value pairs" so that other languages (including JavaScript) can easily read and use it. It is a way to organize information and then transfer it from one location to another using the Internet. The data can be read and inserted into web pages.

You will be using the downloadable JSON file for the remainder of the course as it will provide some sample data for you to use on the web pages you will be building for our weather site.

Three Towns

In the weather.json file you will find weather related data for three towns: Franklin, Greenville and Springfield. For each town you will find data for one day, including: High, Low, Precip and other information. You will want to open the file and take a look at how it is organized, but don't change anything.

Not all at First

We will not use all of the information at first. In fact, you will only use the items as shown in the mobile wireframe for now.
