jQuery: Assessment

The purpose of this week was to get you familiar with basic JavaScript concepts and implementing the jQuery library. As stated earlier it was not to turn you into a jQuery ninja in one week. However, you have been asked to watch a series of videos and follow along with them to gain actual experience writing and using jQuery and build muscle memory while gaining the experience.

Assessment

When done following along with the videos and finishing the code of the individual video files, you are to alter the Lesson4 folder and its files by:

  1. Modifying the folder name to meet the web naming rules.
  2. Comment out the script tags in all html pages that refer to the local version of jQuery.
  3. Create a script tag that uses the CDN path for the current, minified version of jQuery found on jQuery.com's downloads page - https://code.jquery.com/.
  4. Create links to the other html pages in the Lesson4 folder in the footer of the index.htm file of the Lesson4 folder. Style the links using Flexbox so they are horizontal and fit within the footer.
  5. Upload the completed "Lesson4" folder to your web site.

Submission

  1. Create a link to the index page of the Lesson4 folder in your own temporary home page on your web site.
  2. Upload your temporary home page with the new link added.
  3. Once everything is uploaded to your site, test all links to ensure they work!

Grading Matrix

Objective 5 (30 points)

  1. Each web page (6 total) will be checked for operation according to  the finished behavior shown in the Lesson4 videos and as shown below:
    • Index - 3 behaviors: prepend items to the list, validate the form, add a name to the name field if empty.
    • Animated menu - 5 menus slide out from the left.
    • Dropdown menu - submenus drop down from menu bar.
    • Fruit Basket - six boxes are created from a json object.
    • Insert HTML - data is moved between boxes using buttons.
    • Toggle buttons - buttons animate to show they are pressed or not.
  2. Each will be graded "Complete" or not based on their operation.
  3. Each page's completed behavior will be worth 5 points.
