Web Project Overview

The web project is the major project of the semester and is representative of all topics and objectives of the course. The use of code taken from textbooks, video courses and other outside resources is not allowed (although the ideas and concepts from these resources is encouraged). All work must be your own, but you are strongly encouraged to work with your peers.

Project Specifications

During week 12 you will be given the specifications document for the project. From the specifications document you will build your solution. Carefully review the specifications document when it is made available and ask questions as you have them.

Project Submission

The finished project is due at 12:00pm (noon) on the Monday of the last week of the semester (check the course calendar). Follow the submission directions provided in the project specifications file to submit the finished project.

Note: This is a web course intended to help you learn how to construct web sites using HTML5, CSS and JavaScript. Outside templates and frameworks are not allowed.

Grading Matrix

Carefully review the project grading matrix when it becomes available during week 12.

Video Options

The project will require an overview video. For help and options to develop your video consult the Media Help page.
